Page Navigation — 3 Design Approaches

All three replace the 3-mode segmented control with: Start Page picker + Auto Cycle toggle. Same functionality, different visual treatment. Click to open each mockup in a new tab, then select your favorite below.

A — Unified Flat Card

Clean, minimal layout. Start Page dropdown + Auto Cycle toggle with all settings inline. "Pages in Rotation" uses glowing teal pill capsules you tap to toggle. Least visual complexity, most information density.

Best for: Users who want a clean, no-nonsense config panel that shows everything at a glance.

Open full mockup →

B — Visual Page Grid

Pages shown as glassmorphic mini tiles (~60×60px) with icons instead of a dropdown. Selected start page gets a star badge + teal glow. Rotation uses the same tile grid as multi-select. More visual, more tactile — feels like picking from a dock.

Best for: Users who want a visual, touch-friendly interface that feels premium and interactive.

Open full mockup →

C — Carousel Timeline

Start Page as a premium dropdown, then a horizontal timeline strip showing rotation order with arrow connectors between page chips. Loop-back arrow curves from last to first. Chips are clickable to toggle in/out. Most visual representation of the cycle concept.

Best for: Users who want to visualize the actual rotation sequence and page flow.
